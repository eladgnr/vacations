# AWS Deployment Instructions

## Prerequisites
- Docker and Docker Compose installed on AWS EC2 instance
- Ports 8000, 8001, 5173 opened in security groups

## Deployment Steps
1. Copy docker-compose.prod.yml to your AWS instance
2. Copy .env.example to .env and configure with your values
3. Run: docker compose -f docker-compose.prod.yml up -d

## Docker Hub Images
- eladgnr/vacation-web:latest
- eladgnr/vacation-stats:latest  
- eladgnr/vacation-frontend:latest

## Access Points
- Main App: http://your-aws-ip:8000
- Statistics: http://your-aws-ip:5173
- Admin: http://your-aws-ip:8000/admin

## Notes
update in .env:

DJANGO_DEBUG=0 for production
DJANGO_ALLOWED_HOSTS with the actual AWS domain/IP
VITE_STATS_API_URL with the actual AWS endpoint
Strong passwords for database and admin user
Secure secret key for Django
